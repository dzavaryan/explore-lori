Explore Lori — Photo Crop Tool

Click anywhere on a photo to set the focal point. The image adjusts to keep that point centred in the crop.

How to use: 1. Select the crop format (Hero / Card / Square / Wide) that matches where the image is used on the site. 2. Click on the photo to set the focal point (the crosshair shows where). 3. The object-position value updates live — click it to copy. 4. Press Copy All CSS to get a full stylesheet of all current settings.
